⚠️  قبل الاستخدام الفعلي:

  1. غيّر كلمة المرور الافتراضية فوراً
     ✓ بعد الدخول → الملف الشخصي → تغيير كلمة المرور
     ✓ استخدم كلمة مرور قوية (16+ حرف مع أرقام ورموز)

  2. احفظ البيانات في مكان آمن
     ✓ استخدم Password Manager (1Password, Bitwarden)
     ✓ لا تحفظها في ملفات نصية عادية

  3. فعّل المصادقة ثنائية العوامل (2FA)
     ✓ في إعدادات الملف الشخصي
     ✓ أضف رقم هاتفك أو Google Authenticator

  4. لا تشارك البيانات مع أحد
     ✓ هذا حساب خاص جداً
     ✓ يحتوي على جميع صلاحيات النظام

  5. راقب سجل العمليات
     ✓ في قسم Analytique
     ✓ تحقق من العمليات المريبة
